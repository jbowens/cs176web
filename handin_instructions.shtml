Electronic Handin Instructions

  1. Create a folder and place a pdf of your solutions in it.
  2. In a terminal, cd into the folder you created.
  3. Type (or copy and paste) /course/cs176/bin/cs176_handin hwX into the terminal, replacing the 'X' with the homework number. For example, to hand in homework 2, you would type /course/cs176/bin/cs176_handin hw2.
  4. Hit enter to run the script.
  5. You should then receive an email in your CS email account (e.g. jcarrber@cs.brown.edu), notifying you that you've successfully handed in the assignment. If you do not, you should double check the terminal output to make sure the script was run successfully.

If you get this error in part of your terminal output: 

Error was:
    Warning: You are not in the student group - dropping privileges
Check that you are in the cs176studentgroup by running groups in a terminal and checking the output. If you are in the group, log out, then log back in, and try running the handin script again.

If you run into trouble not mentioned here while running this script, see a TA on hours (all hours are posted on the calendar) or email cs176tas@cs.brown.edu. If for some reason you are unable to run the script successfully, and you are unable to reach the TAs, and you have only a few minutes left till the deadline, you may email your solution to cs176tas@cs.brown.edu. However, this should always be your last resort; we reserve the right to deduct points for this kind of handin.
